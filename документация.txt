Документация к ЛР2 

Для создания проекта я использовал среду разроботки QT Creator, его стандартные библиотеки и классы. В первую очередь с помощью QPainter я создал полигон, который показывает цвет палитры. Для плавного изменения цвета, был использован стандартный диалог Qt - QColorDialog, который позволяет как выбирать цвет на палитре, так и плавно изменять его при помощи ползунков. Далее создал Push Button, по нажатию на которую открывается палитра цветов. Создал пять функций,  которые преобразовывают цвет из одной модели в 4 другие. Для пересчета моделей RGB, HSV, HSL, CMYK использовался стандартный класс Qt - QColor. Для пересчета модели XYZ написал формулу пересчета RGB->XYZ и XYZ->RGB. Создал Grid layout, в него поместил девять Push Button, которые отвечают за преобразование цвета из одной модели в две другие. Создал еще один  Grid layout, в него поместил 16 Line edit, которые отвечают за показ каждой характеристики каждой цветовой модели. Приложение создано для Windows. 

При разработке приложения возникла проблема с заданием ограничений значений и их округлением до тысячных долей.
